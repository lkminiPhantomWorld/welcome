# GOVERNANCE — 治理規則

## 什麼能進 public seed

- 公開文件、政策、邊界定義
- 驗證工具
- 授權文件
- Gatekeeper workflow

## 什麼不能進

- LKMini public seed 以外的私有引擎設定內容
- 永恆核心內部規則
- API 金鑰、密碼、個人資料
- 未經驗證的聲明

## PASS / FAIL / UNKNOWN 定義

| 狀態 | 意義 |
|------|------|
| PASS | 可驗證為真 |
| FAIL | 可驗證為假或違反規則 |
| UNKNOWN | 尚未驗證，標記為草稿或推測 |

## 誰能改核心

只有 @ky46738-ops 可以修改核心政策文件。

## 衍生物處理

衍生物必須保留原始 LICENSE、NOTICE 和 attribution。
