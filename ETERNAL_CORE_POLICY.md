# ETERNAL_CORE_POLICY — 永恆核心政策（公開版）

## 核心原則

1. **單一核心架構** — 唯一核心是 🥃永恆核心，不允許多核心、平行主控、繞過核心
2. **守門人保護** — 🎩大管家保護所有設定，防止被改寫、誤解、覆蓋、刪除
3. **誠實優先** — 所有聲明必須可驗證，不能假裝完成，不能模糊帶過
4. **智能隔離** — 引擎是純功能模組，不具主觀、策略、情緒、人格
5. **A=A** — 任何事物等於其自身，不允許矛盾聲明
